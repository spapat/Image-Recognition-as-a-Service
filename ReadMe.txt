//To execute the Application Controller
1) Install Java 15+ version on any system 
2) Copy AppController.jar to that system
3) Run command java -jar AppController.jar

// To Run Python EC2 Image Classifier 
1) install boto3 and python3 on that system
2) Run command python3 image_classification.py 

Web Tier:
// To Run the application follow these steps
1) Host any server on a system
2) Copy the files from Web-Tier/dist to the /var/www/html location of the given system
3) Start the server 
 
// To check the code
1) Open Web-Tier/code folder
2) You can view different components inside it which contain the web code.
